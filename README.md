Este es mi portfolio profesional donde presento mis trabajos como Full Stack Developer. El sitio está diseñado con un enfoque minimalista y "premium", centrándose en soluciones reales que actualmente se encuentran en producción.

🚀 Tecnologías Utilizadas
El proyecto utiliza un stack moderno para garantizar rendimiento y escalabilidad:

Frontend: React 19.

Estilos: Tailwind CSS 4 (Beta).

Herramienta de Construcción: Vite 8.

Deploy: GitHub Actions para despliegue automático en GitHub Pages.

✨ Características Principales
Enfoque en Producción: Los proyectos mostrados no son solo prácticas, sino herramientas que resuelven problemas reales para clientes reales.

Diseño Adaptativo: Experiencia optimizada tanto para móviles como para escritorio.

Navegación Fluida: Implementación de smooth scroll y rutas dinámicas.

Automatización: Integración de flujos de trabajo de CI/CD para actualizaciones rápidas.

🛠️ Instalación y Uso Local
Si deseas ejecutar este proyecto localmente, sigue estos pasos:

Clonar el repositorio:

git clone https://github.com/nolasko7/nolasko7.github.io.git
Instalar dependencias:

npm install
Iniciar el servidor de desarrollo:

npm run dev
Construir para producción:

npm run build
(Los scripts están configurados en el archivo package.json)

📂 Estructura del Proyecto
src/config/data.js: Contiene toda la información personal y de los proyectos.

src/pages/: Contiene las vistas principales como Home y Contacts.

src/hooks/: Lógica reutilizable como el efecto de revelado al hacer scroll.

src/components/: Componentes globales como Navbar y Footer.
